🚀 DEPLOYMENT IN 4 STEPS

Step 1: Environment Setup (5 min)
─────────────────────────────────
  cp .env.example .env
  # Edit .env with:
  # - PRIVATE_KEY (deployer wallet)
  # - POLYGONSCAN_API_KEY

Step 2: Deploy Smart Contracts (10 min)
─────────────────────────────────────────
  npm run deploy:amoy
  
  Output:
  • Deploys CyberNFT contract
  • Deploys CyberNFTMarketplace contract
  • Deploys CyberNFTAuction contract
  • Auto-verifies all 3 on Polygonscan
  • Saves addresses to deployments/amoy.json

Step 3: Configure Frontend (5 min)
──────────────────────────────────
  # Copy deployed addresses to .env:
  NEXT_PUBLIC_NFT_ADDRESS="0x..."
  NEXT_PUBLIC_MARKETPLACE_ADDRESS="0x..."
  NEXT_PUBLIC_AUCTION_ADDRESS="0x..."
  
  npm run generate-constants

Step 4: Deploy Frontend (10 min)
────────────────────────────────
  npm run build
  npm run start
  
  # Test locally, then deploy to:
  # - Vercel (recommended)
  # - Docker
  # - Traditional VPS
