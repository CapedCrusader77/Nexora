IF SOMETHING GOES WRONG
================================================================================

Build Won't Succeed:
1. Clear cache: rm -rf .next node_modules package-lock.json
2. Reinstall: npm install
3. Try again: npm run build

Dev Server Won't Start:
1. Kill existing process: Ctrl+C
2. Wait 5 seconds
3. Try again: npm run dev

Vercel Build Fails:
1. Check Vercel build logs for specific error
2. Verify .env variables are set
3. Check Node version: node --version (should be v18+)
4. Try manual rebuild from Vercel dashboard

Wallet Won't Connect:
1. Check browser console for errors
2. Verify you're on Polygon Amoy testnet
3. Check contract addresses in .env
4. Try refreshing page
